# goosectf-2026
Challenge Repository for GooseCTF (misc0nfig conference CTF) 2026.

## Submitting a Challenge
1. **Fork This Repository**
2. **Copy the misc/example** folder into a new folder, which should be a sub-folder of it's category. E.g. a new pwn challenge called heaps of fun should go in `/pwn/heaps-of-fun`
3. **Write your challenge**. Make sure to include **hosting files** and a **writeup and working solution**
4. **Submit a Pull Request**. A CyberSoc volunteer will confirm your challenge works and **merge it into the repository**
